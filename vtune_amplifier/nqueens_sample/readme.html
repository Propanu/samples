Intel® VTune™ Amplifier nQueens Sample

The nQueens Fortran sample implements a solution to solve an n-queens problem.

Build Instructions

  1. Run the Makefile to clean up all the previous builds as follows:

    $ make clean

  2. Build your target in the release mode as follows:

    $ make

    The nqueens_parallel application is built.

Note:

Detailed build instructions are available in the tutorial listed below.

To use the sample with the VTune Amplifier, make sure to provide an integer value between 4 and 15 as the Application parameters. To follow the tutorial workflow, use the value of 15.

System Requirements

  • Hardware:

    • For the most up to date hardware requirements, see the Intel® VTune™ Amplifier release notes.

  • Software:

    • For the Fortran sample: a Fortran compiler, for example: Intel® Fortran Compiler

Related Tutorials

The nQueens sample for the VTune Amplifier is accompanied by the following tutorials:

  • Finding Hotspots: Identify the most time-consuming program units, analyze thread parallelism on available logical CPU cores, and identify synchronization objects preventing parallelism. This tutorial is based on the nQueens sample located at the \nqueens_fortran folder.
